# **App Name**: GSEnlace

## Core Features:

- Profile Header: Displays the user's avatar, name, subtitle, and a short bio.
- Specialties Section: Showcases the user's areas of expertise with icons, titles, and descriptions.
- Call-to-Action Buttons: Presents primary and secondary call-to-action buttons.
- WhatsApp Button: Links to a WhatsApp chat using wa.me with a placeholder number.
- Footer Section: Displays a minimal footer with a copyright notice and official links.

## Style Guidelines:

- Background: Dark and elegant (#0f0f10).
- Text: Light gray/white for readability against the dark background.
- Accent: Soft brown (#8b6b4f) to highlight interactive elements.
- Font: 'Inter' (sans-serif) for a clean and modern SaaS look.
- Outline icons for specialties to maintain a minimalist aesthetic.
- Mobile-first layout with a maximum width of 420px, centered with padding.
- Subtle hover/press microinteractions for buttons and cards.
