# **App Name**: Link Cinema

## Core Features:

- Link Aggregation: Centralized display of essential links: film spreadsheet, Letterboxd profile, and contact information, with easily editable placeholders for URLs.
- Mobile-First Responsiveness: Responsive design optimized for mobile devices (360px to 430px width), ensuring a seamless app-like experience on smartphones.
- Google Sheet Integration: Link to a Google Sheets document to organize, and show film's info. Display information of +3.000 films
- Theme Customization: Default dark theme with easy color customization for a personalized look.
- Content Loading States: Loading states for avatar and 'Top Picks' list to provide a smooth user experience, even with mock data.
- External Links Handling: Securely open all links in new tabs/windows with `rel='noopener noreferrer'`.
- Top Movie Recommendations: Suggest 'Top Picks of the Week' by automatically grabbing top-rated movies from Letterboxd and creating a summary description of each film using an LLM tool.

## Style Guidelines:

- Background: Dark charcoal (#333333) for a modern, sophisticated feel.
- Primary: Soft beige (#F5F5DC) to highlight key elements and calls to action.
- Accent: Light gray (#D3D3D3) for subtle emphasis and clean separation.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a clean, modern and readable interface.
- Centralized layout with a maximum width of 420px to mimic a mobile app interface, ensuring optimal viewing on various devices.
- Simple, consistent icons for social media links and other actions, enhancing usability.
- Subtle hover and press animations to provide feedback and enhance interactivity without being distracting.
